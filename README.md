# CloudflareAPI_Updater
Bash scripts for auto updating DNS records on cloudflare

# Dependencies

MacOS - `brew install jq`
Ubuntu - `sudo apt install jp`

# Setup

## Make scripts executable

```
    chmod +x setup.sh
    chmod +x cloudflareAPI.sh
```

## Run setup.sh

```
    ./setup.sh
```

Follow script instructions
